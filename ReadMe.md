# npm install
```
$ npm install
```
# Create a .env file
```
PASSWORD=YOUR_PASSWORD
MERGED_PATH=PATH_WHERE_TO_SAVE_MERGED_FILE
DECRYPTED_PATH=PATH_WHERE_TO_SAVE_DECRYPTED_FILE
INPUT_PATH=PATH_WHERE_TO_READ_FROM
```

# Install qpdf
## Ubuntu
```
$ sudo apt-get install qpdf
```
